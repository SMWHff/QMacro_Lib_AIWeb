命令名称： 元素_取属性值
命令功能： 获取网页元素指定属性的信息

WQM命令： Plugin.Web.HtmlGet
命令参数：
参数1【必选】：字符串，网页元素属性类型：
         类型：text，网页元素的内部文本
               html，网页元素的内部HTML
          outerHtml，网页元素的外部HTML
              value，表单元素的value值
                src，图片元素的图片地址
               href，链接元素的链接地址
              class，网页元素的CSS类名称
             offset，网页元素的左上角位置（x|y）
       table[通配符](行,列)，表格元素里的内容，表、行、列是以 0 开始的
       通配符：^，表示对＜thead＞元素的引用，例：table^(0,0)
               $，表示对＜tfoot＞元素的引用，例：table$(0,0)
            数字，表示对＜tbody＞元素的引用，例：table0(0,0)
参数2【必选】：字符串，网页元素特征字符串
                元素特征支持三种格式：
                【WQM  格式】：特征名:特征值&特征名:特征值&特征名:特征值
                【XPath格式】：//*[@特征名='特征值'][@特征名='特征值']
                                带框架：框架索引>>XPath选择器
                              （通过谷歌内核浏览器获取定位路径，按 F12，选中元素，右键 -> Copy -> Copy XPath，就复制到剪切板了。
                                复制到的路径里有双引号，必须全部改成单引号才行）
                【CSS  格式】：*[特征名='特征值'][特征名='特征值']
                                带框架：框架索引>>CSS选择器
                              （通过谷歌内核浏览器获取定位路径，按 F12，选中元素，右键 -> Copy -> Copy selector，就复制到剪切板了。
                                复制到的路径里有双引号，必须全部改成单引号才行）
                                
                    特征名：id，匹配元素唯一标识，可以直接定位
                           tag，匹配元素的标签
                         index，匹配元素的索引，一般在指定特征时无法唯一定位元素时使用，从0开始计算
                          name，匹配元素的name属性
                          type，（支持通配符）匹配元素的type属性
                         value，（支持通配符）匹配元素的value属性，一般用来匹配提交按钮
                           txt，（支持通配符）匹配元素的innerText，一般用来操作链接
                           src，（支持通配符）用于图片元素的匹配，匹配 img 标签的 
                          href，（支持通配符）用于匹配链接的href属性
                         class，（支持通配符）用于匹配元素的CSS类名称
                         frame，框架元素（frame/iframe），frame支持索引匹配和文本匹配，支持嵌套框架索引【frame:0>1>2】
                        通配符：*，表示为特定字符串，  例：value*:一下【匹配所有出现“一下”文字的元素】
                                ^，表示为以某字符开始，例：value^:百度【匹配所有以“百度”文字开始的元素】
                                $，表示以某字符结尾，  例：value$:知道【匹配所有以“知道”文字结尾的元素】

                    特征值：【特征值包含 ":&" 需要先进行转义。 ":" 转为 %3A， "&" 转为 %26】

    mousemove 鼠标被移动。
